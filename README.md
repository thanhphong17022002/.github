# ⚙️ Kho Thiết lập Toàn cục (Global Settings)

Chào mừng đến với kho lưu trữ cấu hình trung tâm! 

Kho `.github` này là một kho lưu trữ **đặc biệt** được sử dụng để quản lý các thiết lập mặc định (Community Health Files) cho **toàn bộ các dự án** thuộc tài khoản GitHub này.

## 🌟 Chức năng của kho lưu trữ này

Thay vì phải copy/paste các file cấu hình lặp đi lặp lại vào từng dự án một, GitHub cho phép sử dụng kho `.github` này làm nơi chứa cấu hình gốc. Bất kỳ dự án nào không có sẵn template riêng thì hệ thống sẽ tự động lấy template từ kho này để áp dụng.

**Các file cấu hình đang được sử dụng:**
- 📄 `PULL_REQUEST_TEMPLATE.md`: Khung sườn tự động hiện ra mỗi khi tạo Pull Request mới. Giúp team review code dễ dàng hơn với Checklist và mô tả chi tiết.
- 🐞 `ISSUE_TEMPLATE/` *(Tương lai)*: Các mẫu báo cáo lỗi (Bug Report) và yêu cầu tính năng (Feature Request) được tiêu chuẩn hóa.

---
*Kho lưu trữ này không chứa mã nguồn (Source Code) của bất kỳ dự án nào.*
